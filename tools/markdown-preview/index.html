使い方

左のエディタにMarkdownを入力すると、右にリアルタイムでプレビューされます。入力テキストはサーバーに送信されません。

ファイルの読み込み

.md / .txt ファイルをエディタエリアにドラッグ&ドロップ、または「ファイルを開く」ボタンから読み込めます。

Mermaidダイアグラム

```mermaid ブロックを使うと、フローチャート・シーケンス図・ガントチャートなどが描画されます。

HTMLダウンロード

「HTMLでダウンロード」でレンダリング済みのHTMLファイルを保存できます。MermaidはSVGで埋め込まれるためオフラインでも表示できます。

リサイズ

中央のバーをドラッグしてエディタとプレビューの表示幅を自由に調整できます。
